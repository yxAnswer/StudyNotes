# 1、linux背景知识

- Linux 有两种含义：
  - 一种Linus编写的开源操作系统的内核
  - 另一种是广义的操作系统

- Linux 版本
  - 内核版本
  - 发行版本

**内核版本**

- https://www.kernel.org/
- 内核版本分为三个部分（例如：5.0.15）
- 主版本号、次版本号、末版本号
- 次版本号是技术为开发板，偶数为稳定版

## 常见目录介绍

- / 根目录
- /root  root用户的 home目录
- /home/username 普通用户的home目录
- /etc 配置文件目录
- /bin 命令目录
- /sbin管理命令目录
- /usr/bin  /usr/sbin 系统预装的其他命令
